## 🎨 设计参数统一

### 背景色
```
网页版：linear-gradient(135deg, #F9F8FF 0%, #FAD0C4 50%, #D9C8FF 100%)
安卓版：BackgroundMain = Color(0xFFF9F8FF)
        GradientPink = Color(0xFFFAD0C4)
        GradientPurple = Color(0xFFD9C8FF)
```

### 玻璃效果
```
网页版：
  background: rgba(255, 255, 255, 0.5)
  backdrop-filter: blur(20px)
  border: 1px solid rgba(255, 255, 255, 0.7)

安卓版：
  GlassFill = Color(0x80FFFFFF)   // 50%透明
  GlassBorder = Color(0xB3FFFFFF) // 70%透明
```

### 文字颜色
```
网页版：
  主文字：#333333
  辅助文字：#8A8A8A

安卓版：
  TextPrimary = Color(0xFF333333)
  TextSecondary = Color(0xFF8A8A8A)
```
