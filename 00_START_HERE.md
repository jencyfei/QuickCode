## 🚀 快速开始（3 步）

### 第 1 步: 安装 APK

```bash
adb install app-release-20251114-v2.apk
```

### 第 2 步: 启动应用

```bash
adb shell am start -n com.sms.tagger/.MainActivity
```

### 第 3 步: 授予权限

首次启动时，点击"允许"授予以下权限：
- ✅ 短信读取权限
- ✅ 通知权限
- ✅ 存储权限

## 📱 功能介绍

### 快递页面 📦

**功能**:
- 按日期分组显示快递
- 支持折叠/展开分组
- 复制所有取件码
- 标记快递为已取

**使用**:
1. 点击底部"📦 快递"
2. 查看按日期分组的快递列表
3. 点击分组标题可以折叠/展开
4. 点击"📋 复制全部"复制所有取件码

### 短信列表 💬

**功能**:
- 显示所有短信
- 按时间倒序排列
- 支持搜索

**使用**:
1. 点击底部"💬 短信"
2. 查看所有短信列表
3. 点击短信可以查看详情

### 标签管理 🏷️

**功能**:
- 查看不同类型的短信
- 按标签过滤短信
- 管理自定义标签

**使用**:
1. 点击底部"🏷️ 标签"
2. 查看所有标签和对应的短信数量
3. 点击标签可以查看该类型的短信

**标签类型**:
- 🔐 验证码 - 验证码短信
- 📦 快递 - 快递通知短信
- 🏦 银行 - 银行相关短信
- 🔔 通知 - 系统通知短信
- 📢 营销 - 营销推广短信

### 设置页面 ⚙️

**功能**:
- 应用设置
- 隐私设置
- 关于应用

**使用**:
1. 点击底部"⚙️ 设置"
2. 选择要修改的设置项
3. 修改后自动保存

## 🐛 常见问题

### Q: 安装失败怎么办？

**A**: 
1. 确保 APK 文件完整
2. 删除旧版本应用
3. 重新下载 APK

### Q: 权限不足怎么办？

**A**:
1. 打开设置 > 应用 > 短信助手
2. 授予所有必要权限
3. 重启应用

### Q: 应用崩溃怎么办？

**A**:
1. 卸载应用
2. 清空应用缓存
3. 重新安装

### Q: 短信无法读取怎么办？

**A**:
1. 确认已授予短信读取权限
2. 确认手机中有短信
3. 检查应用日志
