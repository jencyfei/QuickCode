## 📋 设计概述

基于产品分析文档的推荐方案，生成了一个具有**日期分组 + 日期标签栏 + 快速筛选**功能的新 HTML 版本。

## 🎨 核心功能

### 1. 日期标签栏（顶部）

```
[全部(7)] [11-13(2)] [11-11(5)] [11-07(1)]
```

**特点**：
- ✅ 显示所有可用日期
- ✅ 每个标签显示该日期的快递数量
- ✅ "全部"标签默认选中（蓝色高亮）
- ✅ 其他标签未选中（灰色）
- ✅ 可横向滚动（如果日期很多）
- ✅ 点击标签可快速切换（功能预留）

**样式**：
- 背景：玻璃拟态风格 `rgba(255, 255, 255, 0.5)`
- 边框：`1px solid rgba(255, 255, 255, 0.7)`
- 圆角：`16px`
- 标签颜色：
  - 未选中：灰色背景 + 深灰文字
  - 选中：白色背景 + 蓝色文字 (#667EEA)

### 2. 日期分组显示（中间）

```
▼ 2025-11-13（2件）
  ├─ 快递卡片1
  └─ 快递卡片2

▼ 2025-11-11（5件）
  ├─ 快递卡片1
  ├─ 快递卡片2
  ├─ 快递卡片3
  ├─ 快递卡片4
  └─ 快递卡片5

▼ 2025-11-07（1件）
```

**特点**：
- ✅ 按日期分组显示快递
- ✅ 每个分组有折叠/展开按钮（▼ 符号）
- ✅ 显示分组中的快递数量
- ✅ 默认所有分组都展开
- ✅ 点击分组标题可折叠/展开
- ✅ 折叠时隐藏该分组的所有快递

**样式**：
- 分组标题背景：玻璃拟态风格 `rgba(255, 255, 255, 0.5)`
- 分组标题边框：`1px solid rgba(255, 255, 255, 0.7)`
- 分组标题圆角：`12px`
- 分组标题内边距：`12px`
- 分组标题字体：`14px, 600 weight`
- 折叠按钮：旋转 90 度表示折叠状态

## 🔧 技术实现细节

### HTML 结构

```html
<!-- 日期标签栏 -->
<div class="date-filter-bar">
    <div class="date-tag active">全部 <span class="date-count">7</span></div>
    <div class="date-tag">11-13 <span class="date-count">2</span></div>
    <div class="date-tag">11-11 <span class="date-count">5</span></div>
    <div class="date-tag">11-07 <span class="date-count">1</span></div>
</div>

<!-- 日期分组 -->
<div class="date-group">
    <div class="date-group-header" onclick="toggleGroup(this)">
        <div class="date-group-toggle">▼</div>
        <div class="date-group-info">
            <div class="date-group-date">
                2025-11-13 <span class="date-group-count">（2件）</span>
            </div>
        </div>
    </div>
    <div class="date-group-content">
        <!-- 快递卡片 -->
    </div>
</div>
```

### CSS 关键样式

**日期标签栏**：
```css
.date-filter-bar {
    background: rgba(255, 255, 255, 0.5);
    backdrop-filter: blur(20px);
    border: 1px solid rgba(255, 255, 255, 0.7);
    border-radius: 16px;
    display: flex;
    gap: 8px;
    overflow-x: auto;  /* 可横向滚动 */
}

.date-tag.active {
    background: rgba(255, 255, 255, 0.7);
    color: #667EEA;
    font-weight: 600;
}
```

**日期分组**：
```css
.date-group-toggle.collapsed {
    transform: rotate(-90deg);  /* 折叠时旋转 */
}

.date-group-content.collapsed {
    display: none;  /* 折叠时隐藏 */
}
```

### JavaScript 交互

```javascript
function toggleGroup(header) {
    const content = header.nextElementSibling;
    const toggle = header.querySelector('.date-group-toggle');
    
    // 切换 collapsed 类
    content.classList.toggle('collapsed');
    toggle.classList.toggle('collapsed');
}
```
