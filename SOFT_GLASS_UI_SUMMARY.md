## 🎨 设计参数对比

### 背景色（已调回原始）
```css
/* 原始 + 优化 */
background: linear-gradient(135deg, #F9F8FF 0%, #FAD0C4 50%, #D9C8FF 100%);
```

**参数来源**：
- `background_main`: #F9F8FF (极浅的薰衣草/近白色)
- `gradient_1_pink`: #FAD0C4 (柔和粉)
- `gradient_2_purple`: #D9C8FF (柔和紫)

### 玻璃效果（突出优化）

**卡片填充**：
```css
background: rgba(255, 255, 255, 0.5);  /* 50%透明度 */
backdrop-filter: blur(20px);            /* 20px模糊 */
border: 1px solid rgba(255, 255, 255, 0.7);  /* 70%透明边框 */
```

**参数来源**：
- `glass_fill`: rgba(255, 255, 255, 0.5) (白色, 50%透明度)
- `glass_border`: rgba(255, 255, 255, 0.7) (白色, 70%透明度)
- `backdrop_blur`: 20px

### 阴影效果（微妙）
```css
box-shadow: 0 4px 15px rgba(0, 0, 0, 0.05);  /* 非常微妙 */
```

**参数来源**：
- `shadow`: Extremely subtle shadow (非常非常微妙的阴影)

### 文字颜色

**主文字**：
```css
color: #333333;  /* 深灰色 */
```

**辅助文字**：
```css
color: #8A8A8A;  /* 中度灰色 */
```

**参数来源**：
- `text_primary`: #333333 (深灰色)
- `text_secondary`: #8A8A8A (中度灰色)

## 📊 技术细节

### 玻璃拟态效果公式

```css
/* 基础玻璃效果 */
background: rgba(255, 255, 255, 0.5);      /* 50%透明白色 */
backdrop-filter: blur(20px);                /* 20px背景模糊 */
border: 1px solid rgba(255, 255, 255, 0.7);/* 70%透明边框 */
box-shadow: 0 4px 15px rgba(0, 0, 0, 0.05);/* 微妙阴影 */
border-radius: 16px;                        /* 16px圆角 */
```

### 悬停效果

```css
:hover {
    background: rgba(255, 255, 255, 0.6);  /* 增加不透明度 */
    border-color: rgba(255, 255, 255, 0.8);/* 增加边框可见度 */
    transform: translateY(-2px);            /* 卡片上升 */
    box-shadow: 0 6px 20px rgba(0, 0, 0, 0.08);  /* 增强阴影 */
}
```
