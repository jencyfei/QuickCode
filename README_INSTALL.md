# 短信助手 - 安装与使用指南

**版本**: 1.0.1  
**发布日期**: 2025-11-15  
**APK 文件**: `app-release-20251114-v2.apk`

---

## 🚀 快速开始

### 安装步骤

#### 方法 1: 使用 ADB 安装（推荐）

```bash
# 1. 连接手机
adb devices

# 2. 安装 APK
adb install app-release-20251114-v2.apk

# 3. 启动应用
adb shell am start -n com.sms.tagger/.MainActivity
```

#### 方法 2: 手机直接安装

1. 将 APK 文件传输到手机
2. 打开文件管理器，找到 APK 文件
3. 点击安装
4. 按照提示完成安装

---

## 🔐 权限授予

首次启动应用时，会提示授予以下权限，请点击"允许"：

- **短信读取权限** - 用于读取手机中的短信
- **通知权限** - 用于发送应用通知
- **存储权限** - 用于保存应用数据

如果没有提示，可以手动授予：

1. 打开手机设置
2. 找到"应用" > "短信助手"
3. 点击"权限"
4. 打开"短信"、"通知"、"存储"权限

## 📱 功能介绍

### 快递页面

**功能**:
- 按日期分组显示快递
- 支持折叠/展开分组
- 复制所有取件码
- 标记快递为已取

**使用方法**:
1. 点击底部导航栏的"📦 快递"
2. 查看按日期分组的快递列表
3. 点击分组标题可以折叠/展开
4. 点击"📋 复制全部"复制所有取件码
5. 点击"✓ 全部已取"标记为已取

### 短信列表

**功能**:
- 显示所有短信
- 按时间倒序排列
- 支持搜索

**使用方法**:
1. 点击底部导航栏的"💬 短信"
2. 查看所有短信列表
3. 点击短信可以查看详情

### 标签管理

**功能**:
- 查看不同类型的短信
- 按标签过滤短信
- 管理自定义标签

**使用方法**:
1. 点击底部导航栏的"🏷️ 标签"
2. 查看所有标签和对应的短信数量
3. 点击标签可以查看该类型的短信
4. 点击"⚙️"可以管理规则

**标签类型**:
- 🔐 验证码 - 验证码短信
- 📦 快递 - 快递通知短信
- 🏦 银行 - 银行相关短信
- 🔔 通知 - 系统通知短信
- 📢 营销 - 营销推广短信

### 设置页面

**功能**:
- 应用设置
- 隐私设置
- 关于应用

**使用方法**:
1. 点击底部导航栏的"⚙️ 设置"
2. 选择要修改的设置项
3. 修改后自动保存

## 🐛 常见问题

### Q1: 安装失败

**错误**: `INSTALL_FAILED_INVALID_APK`

**解决方案**:
1. 确保 APK 文件完整
2. 删除旧版本应用
3. 重新下载 APK

### Q2: 权限不足

**错误**: `没有短信读取权限`

**解决方案**:
1. 打开设置 > 应用 > 短信助手
2. 授予所有必要权限
3. 重启应用

### Q3: 应用崩溃

**错误**: 应用启动后立即崩溃

**解决方案**:
1. 卸载应用
2. 清空应用缓存
3. 重新安装

### Q4: 短信无法读取

**错误**: 短信列表为空

**解决方案**:
1. 确认已授予短信读取权限
2. 确认手机中有短信
3. 检查应用日志

### Q5: 快递排序错乱

**错误**: 2024年的快递显示在2025年快递之后

**解决方案**:
- 已在 V1.0.1 版本修复
- 请更新到最新版本

### Q6: 标签过滤不生效

**错误**: 点击标签显示所有短信

**解决方案**:
- 已在 V1.0.1 版本修复
- 请更新到最新版本
