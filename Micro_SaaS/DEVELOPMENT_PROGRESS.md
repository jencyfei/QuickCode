## 🔧 技术细节

### 认证流程
```
用户输入 → 表单验证 → API请求 → Token保存 → 页面跳转
```

### 表单验证规则
**登录表单**:
- 邮箱: 必填 + 格式验证
- 密码: 必填 + 最少6字符

**注册表单**:
- 名字: 必填 + 最少2字符
- 邮箱: 必填 + 格式验证
- 密码: 必填 + 最少6字符 + 大小写字母+数字
- 确认密码: 必填 + 与密码一致

### 错误处理
- 表单验证错误: 显示在字段下方
- API错误: 显示在表单顶部
- 网络错误: 自动重试
