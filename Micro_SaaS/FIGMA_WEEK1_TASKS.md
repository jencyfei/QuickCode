### 任务1.3: 建立排版系统（2小时）

**标题系统**:
```
H1 Display:    Poppins Bold, 32px, 40px行高
H2 Heading 1:  Poppins Bold, 28px, 36px行高
H3 Heading 2:  Poppins Bold, 24px, 32px行高
H4 Heading 3:  Poppins Bold, 20px, 28px行高
H5 Heading 4:  Poppins SemiBold, 18px, 26px行高
```

**正文系统**:
```
Body Large:    Inter Regular, 16px, 24px行高
Body Regular:  Inter Regular, 14px, 20px行高
Body Small:    Inter Regular, 12px, 16px行高
Caption:       Inter Regular, 11px, 14px行高
```

**代码系统**:
```
Code:          Fira Code Regular, 13px, 18px行高
```

**创建方式**:
1. 创建文本框
2. 设置字体、大小、行高
3. 添加标签说明
4. 保存为文本样式

---

### 任务1.4: 建立间距系统（1.5小时）

**间距规则**:
```
XS:  4px
SM:  8px
MD:  16px  ⭐ (标准)
LG:  24px
XL:  32px
2XL: 48px
3XL: 64px
```

**应用规则**:
- 组件内边距: 12px (上下) × 16px (左右)
- 组件间距: 8px (同行) / 16px (不同行)
- 页面边距: 24px (PC) / 16px (手机)

---

## 🖼️ 第二步：高保真页面设计（周二-周三，16小时）

### 页面1: 登录/注册页（3小时）

**PC布局** (1920×1080):
- 左侧 (50%): 渐变背景 (Indigo 600 → 800), Logo, 标题, 特性列表
- 右侧 (50%): 白色背景, 表单, 按钮, OAuth选项

**手机布局** (375×812):
- 全屏表单
- 顶部Logo和标题
- 邮箱、密码输入框
- 登录按钮
- OAuth选项
- 注册链接

**设计要点**:
- 左侧渐变增加视觉层次
- 右侧表单居中
- 按钮使用主色
- 输入框边框#E5E7EB
- 文字对比度满足WCAG AA

---

### 页面2: Dashboard页（3小时）

**PC布局** (1920×1080):
- 顶部导航栏 (56px)
- 欢迎文本
- 快速操作按钮组
- 最近分析卡片 (3列网格)
- 推荐模板卡片 (3列网格)

**手机布局** (375×812):
- 顶部导航栏
- 欢迎文本
- 快速操作按钮 (垂直堆叠)
- 最近分析卡片 (1列)
- 推荐模板卡片 (1列)

**设计要点**:
- 卡片使用白色背景
- 卡片边框#E5E7EB
- 悬停显示阴影
- 网格布局 (3列PC / 1列手机)

### 页面3: 模板选择页（3小时）

**PC布局** (1920×1080):
- 标题和说明
- 分类标签 (水平滚动)
- 模板卡片 (3列网格)
- 自定义模板按钮

**手机布局** (375×812):
- 标题和说明
- 分类下拉菜单
- 模板卡片 (1列)

**设计要点**:

### 页面4: 数据上传页（3.5小时）

**PC布局** (1920×1080):
- 左侧 (50%): 拖拽上传区, 文件预览, 表结构预览
- 右侧 (50%): 指标编辑器, AI建议面板

**手机布局** (375×812):
- 拖拽上传区 (全宽)
- 文件预览
- 表结构预览
- 指标编辑器 (全宽)
- AI建议面板

**设计要点**:
- 拖拽区使用虚线边框
- 中心显示上传图标
- 编辑器使用代码风格
- AI建议显示推荐指标

---

### 页面5: 结果展示页（3.5小时）

**PC布局** (1920×1080):
- 顶部: 标题, 编辑/导出/分享按钮
- 左侧 (60%): 图表展示区
- 右侧 (40%): 关键指标, AI Insights

**手机布局** (375×812):
- 顶部: 标题, 按钮
- 图表展示区 (全宽)
- 关键指标 (全宽)
- AI Insights (全宽)

**设计要点**:
- 图表使用Recharts样式
- 指标显示数值和趋势
- Insights显示分析结果
- 导出/分享按钮可交互
