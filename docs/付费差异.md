# 需求文档：免费版 vs 付费版对比页（HTML实现）

## 1. 页面目的

* **展示免费版与付费版的差异**
* **引导用户付费激活**
* **提高付费转化率**

---

## 2. 页面功能与结构

### 2.1 页面整体布局

* 顶部：页面标题 `免费 vs 付费（激活后）`，字体加粗
* 中部：差异对比表格
* 底部：激活按钮（永久激活 ¥10）
* 可选：顶部可加“流程箭头图”，展示 `免费 -> 激活 -> 完整体验`

---

### 2.2 对比表格设计

| 功能     | 免费版     | 付费版  |
| ------ | ------- | ---- |
| 每日识别次数 | 5次/天    | 不限   |
| 识别速度   | 每条延迟1秒  | 即时解析 |
| 历史记录   | 最近3条    | 全部记录 |
| 批量操作   | 部分记录    | 全部记录 |
| 弹窗提示   | 频繁提示去激活 | 不再提示 |
| UI提示   | 显示限制横幅  | 简洁   |

#### 样式要求

* 免费版列：浅灰背景或提示色
* 付费版列：绿色或亮色，突出可解锁
* 每行可以用图标配合文字（可用 emoji，例如⏱️🗂️🔔）
* 表格边框：轻微圆角，易读性高
* 表格宽度：适应移动端屏幕，横向可滑动（如屏幕宽度不足）

---

### 2.3 页面底部 CTA（Call-to-Action）

* 按钮文本：`立即激活 🔐 ¥10 永久使用`
* 样式：醒目按钮，底部固定或可滑动到视野内
* 点击事件：跳转到激活输入框页面

---

### 2.4 可选增强交互

* **折叠/展开差异详情**：每行差异文字可以展开，初始只显示三条核心差异（次数、速度、记录长度）
* **响应式布局**：移动端优先
* **箭头流程图**（可用 CSS 或 SVG）：`免费 -> 激活 -> 完整体验`

---

### 2.5 HTML实现提示

* 使用 `<table>` 或 `<div>` flex/grid 实现表格
* 免费版/付费版列可用 `class="free"` / `class="paid"` 做样式区分
* CTA 按钮用 `<button>` 或 `<a>` 标签 + JS 点击事件
* 移动端适配：可使用 `meta viewport` + flexbox 或 grid
* 可用 CSS 变量或类控制颜色，方便主题切换

---

## 4. 技术实现建议

   ```css
   .free { background-color: #f5f5f5; color: #333; }
   .paid { background-color: #d4f0d6; color: #000; font-weight: bold; }
   ```
2. **响应式布局**：

   ```css
   @media(max-width: 480px) {
       table { display: block; overflow-x: auto; }
   }
   ```
3. **CTA按钮示例**：

   ```html
   <button class="activate-btn" onclick="goToActivation()">立即激活 🔐 ¥10 永久使用</button>
   ```
4. **可选箭头流程**：

   * 用 `<div>` + CSS `::after` 或 SVG 画箭头
   * 文案：`免费 -> 激活 -> 完整体验`
