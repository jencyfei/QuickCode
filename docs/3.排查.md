# ✅ **《通知无法开启完整排查 Checklist》**

### ① Manifest 必查

* [ ] 声明 POST_NOTIFICATIONS（SDK ≥ 33 必须）
* [ ] 声明 FOREGROUND_SERVICE（若需要后台任务）

### ② Application 初始化

* [ ] 渠道在 Application.onCreate() 初始化
* [ ] 渠道 IMPORTANCE ≥ IMPORTANCE_DEFAULT
* [ ] 渠道 ID 不冲突，未被多处重复创建
* [ ] Application 是否实际生效（检查 AndroidManifest 注册）

### ③ 通知发送验证

* [ ] 手动发送一条测试通知（至少一次）
* [ ] smallIcon 使用合法 vector 图标

### ④ 小米系统特有问题检查

* [ ] 安装来源是否被系统认为“不可信”
* [ ] 应用是否在受限模式
* [ ] 检查系统“显示通知”列表中是否存在 app
