## **1）提升整体视觉层级**

### 原效果

背景渐变饱和，但卡片贴得太紧。

### 优化方案

* page padding 从 20px 提升到 28px
* 卡片 margin-bottom 从 12px 提升到 20px
* 卡片圆角从 12 → 16px
* 阴影增加柔化（0 8px 20px rgba(0,0,0,0.08)）

效果更轻更专业。

---

## **2）图标视觉统一**

统一使用一套 line-style 服务图标，例如：

* 💬（短信）换为线性短信图标
* ✉（通知类短信）换为统一样式 envelope-outline

统一大小：24px
统一颜色：#64748B（冷灰）
统一 stroke-width：1.8px

这一步提升很明显，立刻从“个人开发者工具风”进化为“商业产品风”。

---

## **3）徽标美化（强烈推荐）**

改为：

```
✔ 已授予   → 绿色主题  
⚠ 需手动确认 → 黄色主题  
```

视觉方案：

* 字体：11px
* 高度 22px
* 胶囊圆角：20px
* 内边距：6px 10px
* 色彩搭配：

```
已授予：背景 #ECFDF5，文字 #059669（柔绿）
需确认：背景 #FEF3C7，文字 #B45309（柔橙）
```

专业产品典型样式。

## **4）按钮美化**

原按钮太素 → 优化为：

### 样式：

```
边框色：#CBD5E1  
圆角：12px  
背景：rgba(255,255,255,0.4)  
hover：白 → 微亮  
阴影：0 2px 6px rgba(0,0,0,0.06)
```

### 文案增强：

把按钮文案从：

🔘 前往系统设置
改为：

```
⚙️ 打开系统设置
```

更具操作暗示。
